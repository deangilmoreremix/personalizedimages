## How to Replace Images

### Method 1: Update the Constants File

1. Open `src/constants/imageAssets.ts`
2. Find the image asset you want to replace using its section and slot
3. Update the `url` field with your custom image URL
4. Example:
```typescript
mainFeature: {
  url: 'https://cdn1.genspark.ai/user-upload-image/gemini_generated/your-image-id.png',
  // ... rest of the fields remain the same
}
```

### Method 2: Use the Update Utility Function

```typescript
import { updateImageUrl } from '@/constants/imageAssets';

// Update a specific image
updateImageUrl('hero', 'main_image', 'https://your-cdn.com/new-image.png');
```

### Method 3: Bulk Update via JSON

Create a JSON file with your replacements:

```json
{
  "replacements": [
    {
      "section": "hero",
      "slot": "main_image",
      "url": "https://cdn1.genspark.ai/..."
    },
    {
      "section": "action-figures",
      "slot": "classic_blister",
      "url": "01_Classic_Action_Figure_Modern_Glassmorphism.png"
    }
  ]
}
```
